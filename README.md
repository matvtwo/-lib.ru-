Проект представляет собой серверное веб-приложение, реализующее интеллектуальный HTTP-прокси для исторического литературного архива lib.ru с последующей трансформацией и адаптацией гипертекстового контента под современные пользовательские интерфейсы чтения.

В рамках проекта реализован слой промежуточной обработки HTTP-запросов, который динамически маршрутизирует обращения к различным доменным зонам экосистемы lib.ru (lib.ru, lit.lib.ru, az.lib.ru), корректно разрешая относительные и абсолютные ссылки, восстанавливая иерархию каталогов и предотвращая уход пользователя на внешние ресурсы.

Особое внимание уделено обработке и нормализации текстовых данных, представленных в устаревших кодировках (Windows-1251, KOI8-R), с последующим приведением их к унифицированному UTF-8 представлению. Это обеспечивает корректную визуализацию контента в современных браузерах и упрощает дальнейшую работу с текстом.

Для страниц литературных произведений реализован специализированный режим «читалки», в котором HTML-структура оригинального документа преобразуется в поток чистого текста с сохранением логической структуры, удобной навигации, пагинации и пользовательского управления чтением.

Для каталогов авторов реализована клиентская интерактивная фильтрация, позволяющая выполнять локальный поиск по именам авторов без обращения к серверу, на основе анализа DOM-структуры страницы.

Архитектура проекта демонстрирует практическое применение принципов:
– HTTP-прокси и reverse-proxy,
– серверной трансформации HTML,
– работы с различными кодировками текста,
– DOM-манипуляций на стороне клиента,
– адаптации легаси-контента под современные UX-паттерны.

Проект может рассматриваться как пример интеграции исторических цифровых архивов в современные веб-системы чтения и анализа текста.

Используемые технологии:

Node.js (серверная платформа)

Express.js (HTTP-сервер и middleware-архитектура)

Fetch API (серверные HTTP-запросы)

iconv-lite (декодирование устаревших текстовых кодировок)

HTML string processing (серверная модификация гипертекста)

Vanilla JavaScript (клиентская фильтрация и интерактивность)

CSS (адаптивное оформление и пользовательская читалка)
